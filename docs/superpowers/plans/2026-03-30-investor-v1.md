# Investor V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first working local-first `investor` service that can start a LangGraph run, generate a deterministic research memo, email secure approval links, pause for human review, and resume into an Alpaca review handoff.

**Architecture:** The app is a FastAPI service backed by Postgres for both application data and LangGraph checkpoint durability. A bounded research agent node produces structured recommendations, deterministic services enforce risk and approval policy, and the API layer exposes health, manual trigger, and approval callback routes. The initial delivery stays local-first, with console email and stub-friendly integrations to keep workflow behavior testable.

**Tech Stack:** Python 3.12, FastAPI, LangGraph, SQLAlchemy, Alembic, Pydantic, pytest, Docker Compose, Postgres

```python
def filter_recommendations(items, minimum_conviction, max_ideas):
    seen = set()
    accepted = []
    for item in items:
        if item.conviction_score < minimum_conviction:
            continue
        if item.ticker in seen:
            continue
        seen.add(item.ticker)
        accepted.append(item)
    return accepted[:max_ideas]
```
